Video Learning System（视频学习系统）

📌 项目简介
本项目是一个前端视频学习与互动系统，用于模拟在线学习平台的核心业务流程，包括视频播放、学习进度管理、用户互动及学习数据统计等模块。

项目重点在于实现复杂前端业务逻辑与状态管理，提升组件化开发与工程化能力。

🚀 技术栈
- Vue2 / Vue3
- JavaScript (ES6+)
- HTML5 / CSS3
- LocalStorage
- Axios
- Mock 数据（脱敏处理）

 ✨ 核心功能模块
🎥 1. 视频播放系统
- 基于 HTML5 video 封装播放器
- 支持播放 / 暂停 / 进度拖拽 / 全屏
- 支持播放状态同步
📊 2. 学习进度系统
- 视频学习进度记录
- 断点续播功能（本地缓存 + 数据同步）
- 学习状态可视化（已学 / 未学 / 进行中）

 ❤️ 3. 用户交互系统
- 点赞 / 收藏 / 评论功能
- 支持状态实时更新
- 优化高频交互体验（乐观更新）

📚 4. 学习业务系统
- 每日学习任务（每日一练）
- 学习记录统计
- 学习数据展示（完成率 / 正确率）

🧠 技术亮点（重点！！）
1️⃣ 状态一致性设计
通过 localStorage + 接口数据同步，实现视频播放进度的断点续播功能，保证刷新后状态不丢失。

 2️⃣ 乐观更新优化交互体验
在点赞 / 收藏操作中采用“先更新UI，后请求接口”的方式，提升用户交互流畅度，并支持失败回滚处理。

 3️⃣ 组件化拆分设计
将系统拆分为：
- 视频模块
- 交互模块
- 学习模块
提高代码复用性与可维护性。

4️⃣ 性能优化处理
- 防抖 / 节流优化高频操作
- 列表分页加载优化
- 减少不必要的重复渲染
